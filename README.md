# shapes
Shapes quiz

User can enter text (e.g. Draw a circle with a radius of 100) and the API will decipher and return details to render the shape.

Supported Shapes (properties):
 - Oval (width, height)
 - Circle (radius)
 - Rectangle (width, height)
 - Triangle (side)
 - Square (side)
 - Pentagon (side)
 - Hexagon (side)
 - Heptagon (side)
 - Octagon (side)
